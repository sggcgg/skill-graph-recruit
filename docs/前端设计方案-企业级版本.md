# 🎨 智能招聘分析系统 - 前端设计方案（企业级版本）

> **设计理念**: 科技感 + 简洁专业 + 数据可视化  
> **设计风格**: 符合2025-2026年主流设计趋势  
> **目标**: 提升求职竞争力，展示AI应用能力

> **核心技术架构**:  
> 🤖 Agent对话层: `qwen3.5-plus` API（通义千问）+ LangGraph ReAct Agent  
> 🔍 RAG检索层: ChromaDB向量数据库 + m3e-base语义嵌入  
> 🧠 技能抽取层: 本地部署 `Qwen2.5-1.5B-Instruct`（vLLM加速）  
> 🕸️ 知识图谱层: Neo4j图数据库 + 技能关系网络

```vue
// 按拼音首字母分组（简化：按第一个字母/字符排序展示）
const groupedItems = computed(() => {
  const map = {}
  filtered.value.forEach(item => {
    const key = item[0].toUpperCase()
    if (!map[key]) map[key] = []
    map[key].push(item)
  })
  return map
})
const letters = computed(() => Object.keys(groupedItems.value).sort())

const toggle = (item) => {
  if (selected.value.includes(item)) {
    selected.value = selected.value.filter(i => i !== item)
  } else {
    if (selected.value.length >= props.max) {
      ElMessage.warning(`最多选择 ${props.max} 个`)
      return
    }
    selected.value.push(item)
  }
}
const remove = (item) => { selected.value = selected.value.filter(i => i !== item) }
const scrollTo = (letter) => {
  const el = document.getElementById(`group-${letter}`)
  el?.scrollIntoView({ behavior: 'smooth', block: 'start' })
  activeLetter.value = letter
}
const onScroll = () => {
  // 滚动时高亮当前分组字母
  for (const letter of letters.value) {
    const el = document.getElementById(`group-${letter}`)
    if (el && el.getBoundingClientRect().top >= 0) {
      activeLetter.value = letter
      break
    }
  }
}
</script>
```
